# Repertoire 

**Repertoire** is a premium, all-in-one practice companion designed for musicians who want to master their craft. Built with a focus on aesthetics and usability, it helps you track your song library, manage setlists, and log practice sessions with ease.

##  Key Features

*   **Smart Library**: Organize songs by status (Want to Learn, Learning, Learned) and instrument.
*   **Practice Tools**: Built-in Metronome with haptic feedback, Practice Timer, and Audio Recorder for capturing ideas.
*   **Notes & Progress**: Keep detailed notes for every song and track your mastery percentage.
*   **Setlists**: Create and manage setlists for gigs or practice routines.
*   **Insights**: Visualize your progress with weekly stats, streaks, and activity summaries.
*   **Dynamic Theming**: Beautiful, high-end iOS-style interface that adapts to your preferences.
*   **Power Search**: Integrated with iTunes API to instantly find songs and high-res album art.

## Tech Stack

*   **Framework**: React Native (Expo Router)
*   **Storage**: SQLite (Local Database)
*   **UI**: React Native Paper + Custom Glassmorphism
*   **Audio**: Expo AV

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start the app**:
    ```bash
    npx expo start
    ```
